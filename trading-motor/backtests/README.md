# Backtests

Esta carpeta sirve para documentar pruebas del `CEREBRO` y del motor de decisión.

Orden recomendado:

1. Crear o abrir el archivo del día dentro de `dias/`
2. Anotar cada entrada propuesta
3. Más tarde completar:
   - si ganó o perdió
   - por qué
   - qué regla funcionó o falló
   - qué ajustar

Estructura:

- `INDEX.md`
  - índice rápido de jornadas registradas
- `dias/YYYY-MM-DD.md`
  - un archivo por día
- `TEMPLATE_ENTRADA.md`
  - plantilla base para copiar dentro de cada jornada

Convención recomendada:

- una jornada = un archivo
- varios trades dentro del mismo día
- cierre diario al final con lecciones y métricas
